# KdTree

This work was done as part of a course at the University of Mons. The goal was to implement a data structure called "**Kd-Tree**". This code is design for 2d-Tree. With a higher dimension, there may be issues.

There is a tutorial with all the available features. The text is accessible in the resources of the *main* folder. The code provides four databases, one of them with a dimension of 3.

To make it work, you need at least **java 21** and **gradle**.

The layout designs come from the [uiverse.io](https://uiverse.io/).
